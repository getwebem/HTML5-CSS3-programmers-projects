# Photo Gallery
### Technologies Used
#### HTML5, CSS3, jQuery 3.2, Lightbox.js
 - Run  :  `git clone https://github.com/getwebem/photos.git`
 - Run  :  `cd photos`
 - Run :  `Brackets .`
 - Open :  `Index.html`
 - Browser:  `Live Preview`
